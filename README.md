# C++ 变量与数组动画模拟器

一个面向 C++ 初学者的纯前端教学工具。它可以读取一小段基础 C++ 代码，并把变量、数组、输入输出、条件分支和循环中的值变化拆成可播放的动画步骤，帮助学习者更直观地理解程序执行过程。

## 功能特性

- 可视化展示普通变量和数组的内存格
- 支持逐步执行、上一步、下一步、自动播放和播放速度调节
- 支持 `cin` 输入、`cout` 输出和程序输出预览
- 支持 `if/else`、`for`、`while` 等常见控制结构
- 支持变量声明、数组声明、赋值、自增自减、复合赋值和 `swap(...)`
- 对赋值、交换和数组元素变化提供移动与高亮动画
- 无需安装依赖，直接用浏览器打开即可使用

## 快速开始

克隆或下载项目后，直接打开 `index.html`：

```bash
open index.html
```

也可以在项目目录启动一个本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 使用方式

1. 在左侧代码区输入一段基础 C++ 代码。
2. 如代码中包含 `cin`，在“输入数据”中填入对应输入值。
3. 点击“运行”生成执行步骤。
4. 使用上一步、下一步或播放按钮观察变量与数组如何变化。

项目内置了一个统计数字中 `2` 出现次数的示例，可以点击“示例”快速体验。

## 支持语法

当前解释器聚焦于算法入门和课堂演示常用语法：

- 类型声明：`int`、`long`、`long long`
- 普通变量：`int a = 1;`
- 数组：`int arr[5];`、`int arr[3] = {1, 2, 3};`
- 输入输出：`cin >> a >> b;`、`cout << t;`
- 赋值：`a = b + 1;`、`arr[i] = x;`
- 复合赋值：`+=`、`-=`、`*=`、`/=`、`%=`
- 自增自减：`i++`、`++i`、`i--`、`--i`
- 分支：`if (...) { ... } else { ... }`
- 循环：`for (...; ...; ...) { ... }`、`while (...) { ... }`
- 交换：`swap(a, b);`、`swap(arr[i], arr[j]);`

## 技术实现

项目没有使用框架和构建工具，核心逻辑全部在浏览器中完成：

- `sanitizeSource` 清理头文件、命名空间和 `main` 外壳
- `tokenizeProgram` 与 `parseProgram` 将代码拆解成语句和控制结构
- `executeNodes` 根据语法树模拟执行代码
- `pushFrame` 将每次状态变化保存为可回放的执行帧
- `renderFrame` 与动画函数把变量状态渲染为内存格和移动效果

为了避免死循环或过大的演示任务，执行步数和渲染帧数设置了上限。

## 当前限制

这是一个教学演示解释器，不是完整 C++ 编译器。当前暂不支持函数定义、结构体、指针、引用、字符串变量、浮点数、标准库容器、复杂作用域和完整的 C++ 语法规则。

推荐用于课堂讲解、算法入门、循环与数组过程演示等场景。
