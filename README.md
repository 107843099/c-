# C++ 变量与数组动画模拟器

把一段基础 C++ 代码放进来，看变量、数组、循环和判断一步一步“动起来”。

这个项目是一个纯前端教学小工具，适合用来讲解 C++ 入门算法。它不会试图成为完整编译器，而是专注回答初学者最常问的那几个问题：变量什么时候出现？数组下标指向哪里？`if` 没进的时候发生了什么？循环为什么还在继续？

## 它能做什么

- 把变量和数组画成内存格，值变化时会高亮
- 把数组显示成连续内存条，更容易理解下标和相邻元素
- 把赋值、交换、输入输出拆成可回放的执行步骤
- 用控制流面板展示 `if` 成立、不成立、进入 `else`、循环继续和循环退出
- 自动标记 `for` 迭代器，以及循环体内声明的临时变量
- 支持上一步、下一步、自动播放和播放速度调节
- 内置数字统计、数组排序、前缀和三个教学示例
- 不需要安装依赖，浏览器直接打开就能用

## 怎么玩

1. 在左侧写一小段基础 C++ 代码。
2. 如果代码里有 `cin`，在“输入数据”里填好对应输入。
3. 点击“运行”，生成完整执行过程。
4. 用下一步、上一步或播放按钮观察每一次变量变化。
5. 看右侧控制流面板，判断当前代码是进入分支、跳过分支，还是继续循环。

右上角可以切换内置示例。建议先从“数组排序”开始看，`swap(a[j], a[j + 1])` 的动画很直观。

## 内置示例

- 数字统计：观察输入、取模、计数器和多层循环
- 数组排序：观察数组相邻元素比较和交换
- 前缀和：观察数组输入、递推赋值和下标表达式

## 支持语法

当前解释器面向算法入门场景，支持这些常用写法：

- 类型声明：`int`、`long`、`long long`
- 普通变量：`int a = 1;`
- 数组：`int arr[5];`、`int arr[3] = {1, 2, 3};`
- 输入输出：`cin >> a >> b;`、`cout << t;`
- 赋值：`a = b + 1;`、`arr[i] = x;`
- 复合赋值：`+=`、`-=`、`*=`、`/=`、`%=`
- 自增自减：`i++`、`++i`、`i--`、`--i`
- 分支：`if (...) { ... } else { ... }`
- 循环：`for (...; ...; ...) { ... }`、`while (...) { ... }`
- 交换：`swap(a, b);`、`swap(arr[i], arr[j]);`

## 实现思路

项目没有使用框架和构建工具，所有逻辑都在浏览器里完成。

- 先清理 `#include`、`using namespace` 和 `main` 外壳
- 再把代码拆成语句、代码块、条件和循环
- 然后模拟执行代码，记录每一步的变量状态
- 最后把每一帧状态渲染成内存格、控制流面板和时间轴

为了避免死循环或演示过大，执行步数和渲染帧数都有上限。

## 当前限制

它是教学演示解释器，不是完整 C++ 编译器。暂不支持函数定义、结构体、指针、引用、字符串变量、浮点数、标准库容器、复杂作用域和完整 C++ 语法规则。

换句话说，它适合把“入门算法过程”讲清楚，不适合拿来运行真实工程代码。
